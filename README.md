# Insertion Sort Projesi
#### Insertion Sort'a göre aşamaları:

```
[22,27,16,2,18,6]
[2,27,16,22,18,6]
[2,6,16,22,18,27]
[2,6,16,18,22,27]
```
#### Big-O gösterimi
```
(n*(n+1))/2 = (n^2+n)/2
O(n^2)
```
#### Dizi sıralandıktan sonra 18 sayısı hangi case kapsamına girer?
```
Average Case
```
